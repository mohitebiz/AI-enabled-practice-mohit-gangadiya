### User Acceptance 

- When a user comes to any page, if the current version is not last viewed version then a clear action / button should be visible to user, allowing them to directly check the differences of the document. 
- Once user click to view the difference - a clear highlights should be shown for what has been removed, what content has been added or what are the blocks modified. 
- If the current version is same as the last viewed version, then a clear information or no misleading action should be given to user. 
- As, the feature required underlying sub feature of vX comparison to vZ, the same clear and simple action should be given on the history screen - where user can select any versions they want to compare to. 


### Technical Acceptance
- Edge cases like: No difference in versions should be handled 
- Content modified in between the versions should be handled
- Allow user to mark the version viewed
